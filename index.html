MIME Icon

Site that provides mimetype icons for hotlinking

Usage

This site simply hosts icons for lots of filetypes that you can use and link to. You can use a file extention like /html, a filename like /movie.avi, a mimetype like /application/pdf .

You can also use the ?size=16 to get a consistent size of icons, and the ?default= to force a default instead of an HTTP 404.
